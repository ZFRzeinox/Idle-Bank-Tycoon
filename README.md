# Idle-Bank-Tycoon
Transforme 1 000 € en une immense fortune !

Un petit jeu Idle développé en Python ou le but est de devenir le plus riche possible grâce aux intérêts bancaires.

Vous commencez avec **1 000 €**.

Votre objectif est de gagner de l'argent, Déposer votre argent à la banque, recevoir des intérêts automatiquement, faire des Rebirths pour augmenter votre multiplicateur et atteindre des milliards ...
---

## Fonctionnalités

- Dépôt d'argent
- Retrait d'argent
- Revenus passifs
- Système de Rebirth
- Multiplicateur permanent

---

## Menu

```
1 - Voir l'argent actuel
2 - Déposer de l'argent
3 - Retirer de l'argent
4 - Rebirth
```

---

## Rebirth

Chaque **1 000 000 €** permet d'effectuer un Rebirth.

Chaque Rebirth :

- augmente votre nombre total de Rebirths
- augmente votre multiplicateur de gains
- permet de gagner de l'argent beaucoup plus rapidement
